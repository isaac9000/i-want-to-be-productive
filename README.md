# i-want-to-be-productive
for straight CHUDS who want to minimize dopamine

ANTI-CHUD is a Chrome extension that blocks distracting websites and redirects you when you try to visit them.

## Features

- Block any website by domain name
- Redirects blocked sites to "ANTI-CHUD" page
- "Go Back" button on the blocked page returns you to the last safe URL you were on
- Add and remove sites from the blocklist at any time via the extension popup

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select the project folder
5. The ANTI-CHUD extension icon will appear in your toolbar

## How to Use

**Blocking a site:**
1. Click the ANTI-CHUD icon in your Chrome toolbar
2. Type a domain into the input field (e.g. `youtube.com`, `instagram.com`)
3. Press **Block** or hit Enter
4. That domain is now blocked — any attempt to visit it will redirect to the blocked page

**Unblocking a site:**
1. Open the extension popup
2. Find the domain in the list
3. Click the **✕** button next to it to remove it from the blocklist

**When you hit a blocked page:**
- You'll see the ANTI-CHUD shame screen. Sit and think about your life. 
- Click **Go Back** to return to the last safe page you were on
- Or open the popup and remove the domain from the blocklist if you actually need it
